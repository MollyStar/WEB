## Directory Structures

```
C:\data
    \PSOBB                 // 服务器根
        \SERVER
            \config        // 配置
                \BATTLE    // 战斗数据
                \ITEM      // 物品数据
                \LOGIN     // 登录服务器配置
                \SCRIPT    // 脚本
                \SHIP_1    // 船1配置
                \SHIP_2    // 船2配置
            \LOGIN         // 登录服务器
                \drop      // 掉落表，需要从web下载并复制替换进来，目前没有做link
            \SHIP_1        // 船1服务器
            \SHIP_2        // 船2服务器
    \www                   // web root
        \default
        \pso.codante.org
```
